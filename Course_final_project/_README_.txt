# 目录及运行说明

本目录存放课程设计的所有相关文件。

(以下仅为示范)

## 主要子目录和文件说明

- `data/`
  - 存放测试用的数据。
- `doc/`
  - 存放课程设计报告正文、封面、分工表等文档。
- `src/`
  - 存放源代码。

## 运行说明

所有程序都在命令行中运行，没有图形界面。

### 示范模块

- 显示帮助
  `python example.py -h`

- 运行范例
  `python example.py input.dat ouput.dat`
  输入input.dat，经过高深计算后，将结果写入输出output.dat
